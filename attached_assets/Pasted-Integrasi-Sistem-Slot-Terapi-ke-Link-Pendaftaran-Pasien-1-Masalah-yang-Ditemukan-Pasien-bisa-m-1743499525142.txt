Integrasi Sistem Slot Terapi ke Link Pendaftaran Pasien

1. Masalah yang Ditemukan:
Pasien bisa mendaftar via link, tetapi tidak bisa memilih slot terapi.
Admin sudah mengelola slot di dashboard (Therapy Slots), tetapi belum tersambung ke proses pendaftaran.
2. Solusi yang Dibutuhkan:
a. Tambahkan Form Pemilihan Slot di Link Pendaftaran

Setelah pasien mengisi biodata (nama, tanggal lahir, alamat, nomor WA), tambahkan:

app.post('/api/book-slot', async (req, res) => {
  const { patientId, slotId } = req.body;
  
  // Cek kuota slot
  const slot = await db.query(`
    SELECT * FROM therapy_slots WHERE id = $1
  `, [slotId]);
  
  if (slot.current_count >= slot.max_quota) {
    return res.status(400).json({ error: "Slot penuh!" });
  }
  
  // Simpan ke tabel appointments
  await db.query(`
    INSERT INTO appointments (patient_id, slot_id) VALUES ($1, $2)
  `, [patientId, slotId]);
  
  // Update current_count slot
  await db.query(`
    UPDATE therapy_slots 
    SET current_count = current_count + 1 
    WHERE id = $1
  `, [slotId]);
  
  res.json({ success: true });
});

4. Workflow Penggunaan
Pasien Buka Link Pendaftaran:
Isi biodata (nama, tanggal lahir, alamat, nomor WA).
Pilih tanggal terapi → sistem tampilkan slot tersedia.
Pilih slot → submit.
Admin Dashboard:
Lihat slot terisi secara real-time.
Export laporan harian dengan detail pasien dan slot.

6. Catatan Penting
Pastikan validasi realtime untuk kuota slot (gunakan WebSocket/Firebase).
Tambahkan notifikasi WhatsApp ke pasien setelah booking berhasil.
Uji coba skenario race condition (dua pasien booking slot terakhir bersamaan).
