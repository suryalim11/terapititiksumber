Berikut **saran perbaikan form transaksi** agar bisa mengurangi jatah paket pasien yang sudah membeli sebelumnya:

---

### **1. Tambahkan Opsi "Gunakan Paket Aktif"**  
- Di bawah dropdown **Pilih Paket Terapi**, tambahkan checkbox:  
  ```  
  [ ] Gunakan sesi dari paket aktif (Sisa: X/12)  
  ```  
- Jika dicentang:  
  - Harga terapi otomatis menjadi `Rp0` (karena menggunakan paket).  
  - Sistem mengurangi sisa sesi di database.  

---

### **2. Tampilkan Sisa Sesii Pasien**  
- Saat admin memilih pasien, tampilkan informasi:  
  ```  
  Nama Pasien: John Doe  
  Paket Aktif: Paket 12 Sesi (Sisa: 8/12)  
  ```  

---

### **3. Validasi Otomatis**  
- Jika pasien tidak memiliki paket aktif, hide/disable checkbox "Gunakan Paket".  
- Contoh pesan error: *"Pasien belum memiliki paket aktif."*  

---

### **4. Perbarui Workflow Transaksi**  
```  
1. Admin pilih pasien → sistem cek paket aktif.  
2. Admin centang "Gunakan Paket" → harga terapi = Rp0.  
3. Admin klik "Proses Pembayaran".  
4. Sistem:  
   - Kurangi sisa sesi paket pasien.  
   - Catat transaksi dengan keterangan "Penggunaan Paket".  
5. Invoice hanya mencantumkan produk yang dibeli (jika ada).  
```

### **6. Teknis Implementasi**  
#### **a. Backend Logic**  
- **Endpoint untuk Cek Paket Aktif:**  
  ```javascript
  app.get('/api/patient/:id/packages', async (req, res) => {
    const { id } = req.params;
    const package = await db.query(`
      SELECT * FROM packages 
      WHERE patient_id = $1 AND remaining_sessions > 0
    `, [id]);
    res.json(package);
  });
  ```  

- **Update Sisa Sesii:**  
  ```javascript
  app.post('/api/use-package', async (req, res) => {
    const { packageId } = req.body;
    await db.query(`
      UPDATE packages 
      SET remaining_sessions = remaining_sessions - 1 
      WHERE id = $1
    `, [packageId]);
    res.json({ success: true });
  });
  ```  

#### **b. Frontend (React.js)**  
- **Komponen TransactionForm.js**  
  ```jsx
  function TransactionForm() {
    const [patient, setPatient] = useState(null);
    const [usePackage, setUsePackage] = useState(false);

    useEffect(() => {
      if (patient?.package) {
        setUsePackage(true);
      }
    }, [patient]);

    return (
      <div>
        {/* Pilih Pasien */}
        <select onChange={(e) => setPatient(e.target.value)}>
          <option>Pilih Pasien</option>
          {patients.map(p => (
            <option key={p.id} value={p.id}>
              {p.name} - Sisa Paket: {p.remaining_sessions}
            </option>
          ))}
        </select>

        {/* Opsi Paket */}
        <div>
          <input
            type="checkbox"
            checked={usePackage}
            onChange={() => setUsePackage(!usePackage)}
            disabled={!patient?.package}
          />
          <label>Gunakan 1 sesi dari paket aktif</label>
        </div>

        {/* Produk */}
        <div>
          {products.map(product => (
            <div key={product.id}>
              <input type="checkbox" /> {product.name} - Rp{product.price}
            </div>
          ))}
        </div>

        {/* Metode Pembayaran */}
        <div>
          <button onClick={() => processPayment()}>Proses Pembayaran</button>
        </div>
      </div>
    );
  }
  ```
