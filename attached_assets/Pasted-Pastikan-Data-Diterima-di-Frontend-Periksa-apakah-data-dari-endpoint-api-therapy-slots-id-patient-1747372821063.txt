Query Data Appointment di Server: Cek bagian kode Anda di server, di file server/routes/therapy-slot-patients.ts, yang menjalankan query SQL untuk mendapatkan daftar pasien berdasarkan slot terapi. Pastikan query di file ini mencakup semua status appointment yang sesuai dan mengembalikan hasil yang diharapkan.
// Pastikan query SQL mencakup semua status sesuai
const rawQuery = `
  SELECT 
    a.id as appointment_id,
    a.patient_id,
    a.status,
    p.id as patient_id,
    p.name as patient_name,
    p.phone_number as patient_phone_number
  FROM appointments a
  JOIN patients p ON a.patient_id = p.id
  WHERE a.therapy_slot_id = $1
  AND a.status IN ('Active', 'Booked', 'Confirmed', 'Scheduled') // Validasi status
`;
